Practicing Data Structures and Algorithms daily to improve problem-solving skills and prepare for technical interviews.
# DSA Practice (GeeksforGeeks)

## Day 1
- Problem: Two Sum
- Approach: HashSet (store elements and check complement)
- Time Complexity: O(n)
- Space Complexity: O(n)

---

## Day 2
- Problem: Find Duplicates in Array
- Approach: HashSet (store visited elements, if already present → duplicate)
- Time Complexity: O(n)
- Space Complexity: O(n)

---

## Day 3
- Problem: First Non-Repeating Element in Array
- Approach: HashMap (store frequency of elements, then find first element with frequency = 1)
- Time Complexity: O(n)
- Space Complexity: O(n)
